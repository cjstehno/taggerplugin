Editing a Tag

To edit a tag you will use the buttons available to the Tags View.

  1. In the Tags View, select the tag you want to edit.
  2. Click on the Edit Tag button to open the editor. (You can also double-click on the selected tag to perform the same action)
  3. Enter your changes in the tag editor dialog and click OK or Cancel when you are done to submit or back-out your changes respectively.

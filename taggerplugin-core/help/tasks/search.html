Finding Tagged Resources

Tagged resources can be found using the standard search interface.

  1. Open the search dialog and select the Tags tab.
  2. Select the tags to be used in the search.
  3. Select whether the search is to require that all selected tags be associated with a resource or just any for a valid match.
  4. Select the scope of the search.
  5. Click the Search button. The results will be displayed in the standard search results view.
  6. A resulting resource can be double-clicked to open its associated editor.

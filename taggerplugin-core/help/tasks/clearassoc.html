Removing Tag Associations

Tag Associations are removed through the context menu.

Removing All Associations

  1. Right-click on the selected resource or resources to open the context menu.
  2. In the Tags submenu, select the "Clear Tag Association(s)" option to remove all tag associations from the selected resources.
  3. If specified in the preferences, you will be asked for confirmation of the removal.

Removing Selected Associations

  1. Right-click on the selected resource or resources to open the context menu.
  2. In the Tags submenu, select the "Remove Tag Association(s)" option to open the Tag selection dialog. This dialog will show only tags associated with at least one of the selected resources.
  3. Select the tag associations to be removed and click OK.
  4. If specified in the preferences, you will be asked for confirmation of the removal.
